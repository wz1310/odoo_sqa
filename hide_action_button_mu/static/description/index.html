Hide The Action Button For The Particular User

Hide Action Menu From all the Tree View and Form View for the specific group of Users.

Features

This module hide the Action button for the particular user

Configuration

Hide the action button for a particular user so that, you have to go to Settings --> Users & Companies --> Users then open the user and mark the "Hide Action Button"(You can see this configuration after activate the debug mode.).

Hide Action Button For Particular User

In Tree View, Hide the action button.
e.g. Quotations

In Form View, Hide the action button.
e.g. Quotations

In Tree View, Hide the action button.
e.g. Customer Invoices

In Form View, Hide the action button.
e.g. Customer Invoices
